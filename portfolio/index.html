    Este proyecto tiene como objetivo crear un conjunto de herramientas para administrar almacenes de vectores de manera agnóstica al proveedor. Esto permite realizar migraciones, consultas y visualizaciones utilizando diferentes técnicas de reducción para vectores densos, entre otras funcionalidades.
